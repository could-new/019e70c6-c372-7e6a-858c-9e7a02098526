# Skill Selector App

A beautiful, responsive Flutter application that allows users to choose their starting skill (e.g., Programming, Design, Writing, Marketing). It adapts smoothly across mobile, tablet, and desktop layouts.

## Features
- Interactive grid selection of skills
- Responsive layout (1 column on mobile, 2 on tablet, 4 on desktop)
- Animated selection states and color highlights
- Modern Material 3 UI

## Getting Started

1. Ensure you have Flutter installed.
2. Clone this repository.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to start the app.
